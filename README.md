# Лабораторная работа №4: Многопоточность

## Описание
Проект содержит три задания по многопоточному программированию на C++.

### Задание 1
Сравнение примитивов синхронизации:
- Mutex
- Semaphore
- Barrier
- SpinLock
- SpinWait
- Monitor

### Задание 2 (Вариант 17)
Многопоточная фильтрация автомобилей по критериям:
- Диапазон цены
- Максимальный пробег
- Минимальный год выпуска

### Задание 3 (Вариант 1)
Реализация алгоритма банкира Дейкстры для предотвращения дедлоков.
